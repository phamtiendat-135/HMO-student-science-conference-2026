# 📋 Templates - Mẫu báo cáo tóm tắt

## Hướng dẫn sử dụng

1. Tải file mẫu abstract tại thư mục này
2. Điền đầy đủ thông tin theo mẫu
3. Lưu file dưới định dạng PDF hoặc DOCX
4. Nộp qua Google Form trước hạn chót **20/03/2026**

## Quy cách trình bày

- **Độ dài:** Tối đa 2 trang A4
- **Font chữ:** Times New Roman, cỡ 12
- **Giãn dòng:** 1.5
- **Lề:** Trên/Dưới: 2.5 cm, Trái: 3 cm, Phải: 2 cm
- **Ngôn ngữ:** Tiếng Việt hoặc Tiếng Anh

## Cấu trúc bài tóm tắt

1. **Tiêu đề** (Title)
2. **Tác giả và Đơn vị** (Authors and Affiliations)
3. **Giáo viên hướng dẫn** (Supervisor)
4. **Tóm tắt** (Abstract) - 200-300 từ
5. **Từ khóa** (Keywords) - 3-5 từ khóa
6. **Tài liệu tham khảo** (References) - nếu có

## Lưu ý

- Giảng viên hướng dẫn sẽ upload template file (.docx) vào thư mục này
- Sinh viên vui lòng kiểm tra thường xuyên để tải mẫu mới nhất
