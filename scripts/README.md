# Scripts - Cong cu xu ly du lieu

## Mo ta

Thu muc nay chua cac script ho tro xu ly du lieu cho hoi nghi, bao gom:

- Xu ly va tong hop danh sach dang ky
- Thong ke bai bao cao theo linh vuc
- Tao lich trinh tu dong
- Cac cong cu ho tro khac

## Ngon ngu ho tro

- Python (.py)
- R (.R)
- MATLAB (.m)

## Huong dan su dung

1. Clone repository ve may
2. Cai dat cac thu vien can thiet (neu co)
3. Chay script theo huong dan trong tung file

## Cau truc thu muc

```
scripts/
├── README.md          # File nay
├── data_processing/   # Script xu ly du lieu (se cap nhat)
├── statistics/        # Script thong ke (se cap nhat)
└── utils/             # Cac tien ich chung (se cap nhat)
```
